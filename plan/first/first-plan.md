# Minecraftサーバー管理アプリ作成メモ - 初期版

複数台のサーバーマシン(WindowsまたはUbuntuServerまたはMac)を所有しており、
Minecraftゲームサーバーを簡単にホストし、制御/監視するためのツールを作成しようと考えている

* LAN内のWindowsやLinuxのマシン上でMinecraftサーバーを簡単に実行できる(LAN外は想定しない)
* サーバーコンフィグ書き換え、起動/停止、ステータス確認が遠隔で行える
* 管理コンソール(今回のUIモックアップ)はElectronを使い、Win/Linux/Macで動作する
* サーバーエージェント(各物理マシンで動作)もWin/Linux/Macで動作する
* どちらも消費リソース目標は特に定めないが、管理コンソールの操作はストレスを感じない程度に軽量にする
* ポート開放は手動で行うものとするが、サーバーごとのポート割り当てを補助する機能を設ける
* 管理コンソールは物理サーバーとゲームサーバーの両方を個別で監視/管理できるようにする
* 管理コンソールとサーバーエージェントの両方が同じマシンで実行される可能性がある（ただし特別扱いはしない）
* サーバーエージェントと管理コンソールはws(非暗号化WebSocket)で常時接続し、リアルタイムなステータス確認やコマンド送信を可能にする
* サーバーエージェントは管理コンソールに手動で登録される（ローカルIPを使用）
